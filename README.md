# Tourism_NLP
## 软件要求
1. snownlp
2. pandas
3. jieba
4. 建议下载jupyter,通过清华镜像源安装会快一些  
命令
先更新一下pip  
`pip install --upgrade pip
`  
然后执行安装命令  
`pip install -i https://pypi.tuna.tsinghua.edu.cn/simple jupyter`

然后终端输入`jupyter notebook
`查看能否正常使用  
基本使用参见博客  
https://blog.csdn.net/qq_33619378/article/details/83037106
